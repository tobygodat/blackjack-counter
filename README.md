# Blackjack Card Counter

School project. A simple Hi-Lo card counter for blackjack.

## How to use

Open `index.html` in a web browser. Click a card button each time you see one dealt. The running count and true count update automatically.

## Hi-Lo values

- 2, 3, 4, 5, 6 = +1
- 7, 8, 9 = 0
- 10, J, Q, K, A = -1

True count = running count / decks remaining.
