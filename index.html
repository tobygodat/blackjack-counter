Hi-Lo values

  • +1 2, 3, 4, 5, 6 — low cards help the dealer. Gone from the shoe ⇒ player edge rises.
  • 0 7, 8, 9 — neutral, ignored.
  • −1 10, J, Q, K, A — big cards help the player (blackjacks, dealer busts).

True count

The running count is noisier in deep shoes. Normalise:

true = running ÷ decks remaining

The true count is what actually predicts edge — each +1 is worth roughly 0.5% player advantage.

Bet spread

A common starter ramp (1 unit = your min bet):

  • TC ≤ 1 1 unit
  • TC 2 2 units
  • TC 3 4 units
  • TC 4 6 units
  • TC ≥ 5 8 units

Shortcuts

  • 2–9 add that card
  • 0 or T adds a ten
  • J Q K A add face/ace
  • U undo · R reset shoe
  • [ ] change shoe size

What this is not

This is a training tool. It doesn't track casino rules, insurance, deviations, or the Illustrious 18 — it's the Hi-Lo count, a deck estimator, and a simple spread. Pair it with basic strategy before anything else.
